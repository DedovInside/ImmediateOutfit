# ImmediateOutfit Roadmap

## Next
- Прогнать live QA по боту и dashboard на реальных пользователях.
- Проверить DeepSeek text-AI на сценариях "Проверить мой образ" и "Подобрать под мою вещь"; сравнить качество с rule-based fallback.
- Добавить реальные картинки/URL к командным образам, если команда передаст материалы.
- Усилить сценарий "несколько дел за день" и замену отдельного элемента образа.
- Провести 2-3 коротких пользовательских спринта и собрать изменения метрик.

## Semester
- Доработать photo-review как отдельный эксперимент после проверки core value.
- Протестировать Qwen/GigaChat/Gemini vision API для photo-review, если текстовый AI покажет спрос.
- Ввести более точную сегментацию пользователей по стилю и бюджету.
- Протестировать недельные подборки как premium-функцию.
- Протестировать Telegram Stars для цифрового Premium без карточного эквайринга в боте.

## Later
- Mini app для визуального каталога гардероба.
- Автоматическая работа с фото вещей.
- Полноценная оплата, возвраты, support flow и коммерческий пакет.
- Pivot в сторону AI outfit review, если именно этот сценарий покажет лучшие метрики удержания.
