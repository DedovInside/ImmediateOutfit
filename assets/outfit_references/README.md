# Outfit References

Каждая подпапка соответствует `outfit.id` из `data/team_outfits.json`.

Как добавить картинки:

1. Найди нужный образ в `data/team_outfits.json`.
2. Открой папку `assets/outfit_references/<outfit_id>/`.
3. Положи туда изображения референсов.

Поддерживаемые форматы:

- `.jpg`
- `.jpeg`
- `.png`
- `.webp`

Бот отправляет до 10 изображений из папки по кнопке `Референс`.
Имена файлов могут быть любыми, но удобнее использовать:

```text
01.jpg
02.jpg
03.png
```

Для старых образов папки не создаются. Если картинок нет, бот показывает только текстовый референс.
